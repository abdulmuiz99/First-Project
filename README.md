# Power BI Data Modeling & Dashboard Project

This project demonstrates an end-to-end Business Intelligence workflow using Power BI, covering the complete process from raw data preparation to interactive dashboard development.

# Skills Demonstrated

Data Cleaning using Power Query
Data Transformation and Shaping
Data Type Optimization
Creating Relationships between Tables
Data Modeling using a Star Schema approach
Building a Calendar Table
Creating Date Hierarchies
Writing DAX Measures
Creating Calculated Columns
Creating Custom Tables
Building Interactive Dashboard Visualizations
Applying BI Best Practices for Data Analysis
Key Learning Outcomes
Prepared raw datasets for analysis through data cleaning and transformation.
Designed an efficient data model by establishing relationships between multiple tables.
Created reusable DAX measures for analytical calculations.
Implemented calculated columns and custom tables to enhance reporting capabilities.
Built a calendar table with hierarchies to support time-based analysis.
Developed an interactive dashboard for exploring geographical and product-related information.

# Tools Used

Power BI Desktop
Power Query
DAX (Data Analysis Expressions)

This project was built as part of my journey toward becoming a Business Intelligence (BI) Developer, with a focus on developing strong skills in data modeling, data transformation, DAX, and interactive dashboard design.
